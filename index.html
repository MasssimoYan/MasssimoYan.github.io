HMI 規格與功能

HMI 設備廠牌與型號
  • 設備廠牌: 北爾人機
  • 設備型號: iX Base 15 v2
  • 具備X2 Web，內建HTML5 Web Client
  • UL、CE、FCC 與KCC 認證整個 X2 系列均通過嚴苛的認證標準。
  • 寬溫操作範圍-10°C 至 +60°C寬溫操作範圍-10°C 至+60°C (X2 pro web)和-30°C 至+70°C (X2extreme web) 的廣泛工作溫度可滿足大部分產業的需求。
  • Cortex-A9 處理器強大的單核心或雙核心ARM Cortex-A9 處理器在高需求應用方面提供快速的畫面切換與程式執行功能。
HMI 通訊功能
  • 與PLC 通訊: 透過Modicon通訊，與台達 AH-PLC 網路模組交握，數量:3台
  • HTML5 網頁建置:建置HTML5網頁功能，讓使用人員透過網頁操作現場的閘門，因需跨裝置顯示，須將網頁設計為響應式網頁
  • 農水署平台:與農水署平台是透過OPCUA傳輸Tag狀態與控制，需參照農水署的文件建置Tag名稱與資料格式。


HMI 軟體系統架構

    • 權限管控:分觀看、操作、管理三個等級

雲端 規格

雲端介接與通訊協定

  • 通訊協定: OPC UA、 Mqtt
  • api 介接: Web Api

問題彙整

  • 依照設計所需的plc模組數量與單價分析表的數量不一，需要增加DI、 DO 模組，且部分模組從其他箱體移過來。
  • 因大目提供的開度為可變電阻，因此需要變更CM5H的型號!!!
  • 雲端平台的架構，因無相關規範，需要進行需求訪談，訪談後再進行規劃。
  • 雷達波水位計數量僅有一組，依據業主需求須於IPC2-1 加裝一組水位計(需在採購)
  • 突波抑制器為兩組，分別在MPA、IPC2-1內。

注意事項

PLC

  • 台達的PLC模組數量，需要再確認，與標單的數量不一樣

HMI

  • 北爾人機 非標單內設備，需特別注意順大箱體設計是否與前期規劃的吻合。
  • HMI系統流程圖，需規畫繪製後提供給政佑、亞聖。

大目機械

  • 大目機械，開度的可變電阻規格:RV24YN 20S B103 10KΩ

雲端平台

  • 該項目除了規範有寫需進行原始碼的弱點掃描外，其餘的部分，設計單位、監造單位、使用單位皆無想法。
  • 預計於12/2 軟體部朝開會議討論雲端平台施作的項目。
  • 雲端平台的製作，由資識執行，新陳規劃。

水位計

  • 雷達波水位計: 路斯-Sitrans LR150，數量:1組
  • 另一組水位計: 預計採用超音波水位計，路斯 Honda HD353A，數量:1組
